# Day 1 – Introduction to SRE

## Hands-On: Navigating OCI Console (Monitoring & Compute)

---

## Step 1: Open the OCI Console

1. Open your web browser.
2. Visit: [https://cloud.oracle.com](https://cloud.oracle.com)
3. Sign in using your OCI account credentials.

**Why this matters:**
The OCI Console is where you will access all cloud resources during the course.

---

## Step 2: Locate the Compute Service

1. Open the **Navigation Menu** in the top-left corner.
2. Scroll to the **Compute** section.
3. Click **Instances**.

**What this service is:**
Compute provides virtual machines that run applications, tools, scripts, and workloads.

You should now see a list of instances (or an empty page if none exist).

---

## Step 3: Locate the Monitoring Service

1. Open the **Navigation Menu** again.
2. Scroll to **Observability & Management**.
3. Click **Monitoring**.
4. Select **Metric Explorer**.

**What this service is:**
Monitoring provides built-in metrics such as CPU usage, memory usage, and network performance.

You should now see a dashboard where metrics can be selected and viewed.

---

# Notes

* These two services—Compute and Monitoring—will be used repeatedly throughout the course.
* Make sure you know where they are located in the menu.
* No configuration changes are needed for this exercise.
